How To Backup Mikrotik - Mikrotik Script

How To Backup adn Restore on Mikrotik RouterOS

# backup all script to (file .rsc) 
/exp file=namafile

# backup all script to (file .backup) 
/system backup save name=namafile.backup

# If you want to protect and add a password during restore
/system backup save name=[namafile] password=[password]

# How to restore a backup file is quite easy
/system backup load name=[namafile]
