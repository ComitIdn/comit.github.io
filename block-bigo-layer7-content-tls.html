How to Block Bigo with "Layer 7" or "Content" Or "TLS"

Block Bigo With "Layer-7"

/ip firewall layer7-protocol
add name=Telegram regexp="^.+(bigo.tv).*\$"
/ip firewall filter
add action=drop chain=forward layer7-protocol=Telegram

Block Telegram With "Content"

/ip firewall filter
add action=drop chain=forward content="bigo.tv" 
add action=drop chain=forward content=".bigo."

Block Telegram With "TLS"

/ip firewall filter
add action=drop chain=forward protocol=tcp tls-host="bigo.tv"
add action=drop chain=forward protocol=tcp tls-host="*.bigo.*"
