Change Name Interface or Ether To Default Manua - Mikrotik Script

I made this simple script to make it easier for us to return the interface name (ether) to the factory default name without having to reset the proxy. There are times when we forget the order on the Mikrotik port even though we have given the ether name, for example with the name "LAN" but whether "LAN" is plugged into what port. so the meaning is clear :)

#####################################
# Script by Agus Ramadhani
# fb.com/buananet.pangkalanbun
# http://www.o-om.com
# SCRIPT name interface to default
######################################
/interface ethernet
set [ find default-name=ether1 ] name=ether1
set [ find default-name=ether2 ] name=ether2
set [ find default-name=ether3 ] name=ether3
set [ find default-name=ether4 ] name=ether4
